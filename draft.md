## API
```sh
## Changing theme
set -g @shifter_theme 'everforest'
## Changing window layout
set -g @shifter_window_modules '#name'
set -g @shifter_window_modules '#number:  #name'

## Adding separator
set -g @shifter_aside_separator '<'

## Aside modules
##  #username | #hostname | #session-name
set -g @shifter_aside_modules '#date-and-time #session-name'
```
